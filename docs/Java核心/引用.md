<h4>Java中的引用</h4>
从JDK1.2开始，Java对引用的概念进行了扩充，将引用分为强引用、软引用、弱引用和虚引用四种。<br/>
<b>强引用</b>(Strong Reference)：是在代码种普遍存在的，例如：<code>String str = "a"</code>表达式中，变量str对字符串常量对象的引用就属于强引用(由于String是一个比较特殊的类型，属于不可变的常量类型，所以实际上<code>Object o = new Object()</code>作为例子会更合适)。由此可见，强引用是我们平时所用最多、最熟悉的一种引用。如果一个对象的引用方式属于强引用，垃圾回收器(GC, garbage collector)将不会对改对象进行内存回收，即使堆中内存不足、抛出OOME(OutOFMemeryError)。
